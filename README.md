REQUIREMENTS
------------
* Docker & Docker Compose
* Make

INSTALLATION
------------

### Install via Make

First you need to build the project
```
make build
```
After you should to run docker environment
```
make run
```
Now project accessible at `http://localhost:8000`

Also you can run tests
```
make tests
```
To run static analyse
```
make phpstan
```

Enjoy!
